# Create Antlr4
Using interactive command line to quickly create an ANTLR4 project.


## Create An ANTLR4 Project
With NPM:

```bash
$ npm create antlr4
```

With Yarn:

```bash
$ yarn create antlr4
```

With PNPM:

```bash
$ pnpm create antlr4
```

## Built-in Templates
+ Antlr4ng + TypeScript (Recommended)
+ Antlr4 + JavaScript
